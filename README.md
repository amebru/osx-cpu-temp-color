# osx-cpu-temp-color
CLI color-coded auto-updating cpu temperature monitor for your mac based on [osx-cpu-temp](https://github.com/lavoiesl/osx-cpu-temp).


## Installation
```
brew install osx-cpu-temp
git clone <repo-url>
cd osx-cpu-temp-color
chmod u+x osx-cpu-temp-color
```

You might want to put this somewhere on your PATH:
```
ln osx-cpu-temp-color ~/.local/bin
```

## Usage
```
% osx-cpu-temp-color 
04:58:08 58.6°C   
```
