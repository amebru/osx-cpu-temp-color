# osx-cpu-temp-color
CLI color-coded auto-updated cpu temperature monitor for your mac

```
git clone <repo-url>
cd osx-cpu-temp-color
chmod u+x osx-cpu-temp-color
```

You might want to put this somewhere on your PATH:
```
ln osx-cpu-temp-color ~/.local/bin
```

Then run it:
```
% osx-cpu-temp-color 
04:58:08 58.6°C   
```
